The following visualization shows county by county numbers (quantity) of emergency service providers for COVID-19 related endeavors. This data was sourced independently from the ArcGIS Hub open data and linked directly to Tableau using Javascript and html to produce this web data connector whose button is clickable within the Tableau environment.

More importantly, it is interesting to note that Los Angeles and Orange Counties have the highest number of said providers with a total of 1,025 and 388 respectively.
